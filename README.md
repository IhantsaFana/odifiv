# fivondronana
Application mobile de Digitalisation du Fivondronana

# C'est quoi fivondronana ?
Au sein du scout protestant à Madagasikara (SAMPATI ou FSFLM ou SATIMPA), il y a l'hierarchi :
- Foibe
- Faritany
- Faritra
- Fivondronana
- Sampana

Un fivondronana doit être dans un Faritra dans un Faritany, il a un nom et un numéro identifiant par exempla 420.

Dans chaque fivondronana, il y a : 
- Mpanazava (fille)
- Tily (garçon)

Membres : 
- Filoha
- Tonia
- Mpiandraikitra
- Beazina

Pour les mpanazava, il y a 3 Sampana : 
- Voronkely (6 à 12 ans) - Couleur : Mavo (Jaune)
- Mpanazava (12 à 16 ans) - Couleur : Maitso (Vert)
- Afo (+16 ans) - Couleur : Mena (Rouge)
- et les Mpiandraikitra ou bien les responsable de chaque Sampana

Pour les Tily :
- Lovitao (6 à 12 ans) - Couleur : Mavo
- Tily (12 à 16 ans) - Couleur : Maitso
- Mpiandalana (16 à 21 ans) - couleur : Mena
- Menafify (+21ans) - Couleur : Grenat
- Les Mpiandraikitra pour chaque sampana

Mpiandraikitra :
- Tily/Mpanazava
- Sampana (Couleur)
- Andraikitra (ex: Mpampilalao/Mpandrindra/Tompo-toerana)

Chaque membre : 
- nom et prénom
- totem ou prénom préféré
- photo
- adresse
- date : 
* nidirana skoto
* nanaovana fanekena (avec lieu)
* fitoizan-tsampana (liste car il y a 3 ou 4 sampana) (C'est la date ou il/elle a monté du Sampana quand il/elle a l'âge)
- talenta
